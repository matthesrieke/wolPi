wolPi

Android App for Wake-on-Lan via SSH (e.g. Raspberry Pi, Debian-based remote machines)

Download as .zip Download as .tar.gz View on GitHub

"My stupid router is blocking the broadcast IP!"

Yes, mine too. So I am using a Raspberry Pi as a proxy to send my magic packet call.

This is a convenience app, providing a one-click WOL call (via SSH and the 'wakeonlan' module).
